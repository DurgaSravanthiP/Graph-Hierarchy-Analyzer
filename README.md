# 🌳 Graph Hierarchy Analyzer

A full-stack web application that parses directed edge relationships, detects cycles, and visualizes hierarchical tree structures in real time.

---

## 🚀 Live Demo

🔗 Frontend: https://graph-hierarchy-analyzer.vercel.app  
🔗 Backend API: https://graph-backend-gqoe.onrender.com

---

## 📌 Features

- ✅ Parse directed edges (e.g., A->B, B->C)
- 🔍 Validate input data
- ⚠️ Detect cycles in graphs
- 🌳 Generate hierarchical tree structures
- 🔁 Identify duplicate edges
- ❌ Detect invalid entries
- 📊 Summary insights (trees, cycles, largest root)
- 🎨 Interactive and modern UI
